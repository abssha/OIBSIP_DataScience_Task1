**Title** - Iris Flower Classification

## Objective 
Build a machine learning model to classify iris species based on flower measurements.

## Dataset
UCI Iris Dataset (150 samples, 4 features, 3 classes)

## Steps Performed
Data loading
Data cleaning
Label encoding
Train-test split
Feature scaling
Model training (KNN, Logistic Regression)
Model evaluation
Model saving

## Tools Used
Python
Pandas
NumPy
Scikit-learn
Matplotlib
Seaborn
VS Code

## Outcome
Achieved 100% accuracy on test data using KNN classifier.
